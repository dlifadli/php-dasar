1. Kenapa Harus Memilih PHP?
- Relatif mudah untuk pemula
- Pre-requisite HTML + CSS
- Gratis & open source (php.net/downloads.php)
- Dukungan komunitas yang besar (Google = 9,300,000,000 sources & stackoverflow = 1,090,144 sources & facebook group = PHP INDONESIA 145rb+ member)
- Dokumentasi yang lengkap (http://php.net/manual/en/)
- PHP dibuat khusus untuk web
- Kebanyakan website yang ada saat ini dibuat dengan PHP
- Banyak website besar dibuat dengan PHP (facebook, yahoo, wikipedia & flikr)
- Banyak CMS(Content Management System) yang menggunakan PHP (wordpress, joomla, drupal & moodle)
- Banyak Framework PHP (CodeIgniter(CI), CakePHP, yiiframework, laravel & Symfony)

2. Apa Yang Akan Dipelajari?
- Client-side VS. Server-side Scripting
- Static VS. Dynamic Website
- Persiapan Lingkungan Pengembangan
- Sintaks PHP
- Array
- Request Method
- Studycase Website Sederhana
- Databases
- CRUD (Create, Read, Update, Delete)
- Login & Registrasi
- Session & Cookie
- ajax (Asynchrounus Javascript XML)
- Upload File
- PDF Reporting (Print/Cetak)
- Web Hosting
